Projet API First

Votre fil rouge : Développez un projet collaboratif avec l'approche API First

🎯 Objectif du projet

Conception et développement d'une application complète en utilisant l'approche API First. Vous travaillerez en équipes mixtes (frontend/backend) avec une coordination optimale grâce aux outils et méthodes enseignés.

🏗️ Architecture cible

API REST avec OpenAPI
Tests automatisés
Frontend moderne
Déploiement continu

📦 Technologies utilisées

OpenAPI

Git & GitHub

Workflow collaboratif et CI/CD

Tests

Jest, Cypress, Postman

Docker

Conteneurisation et déploiement

📋 Progression Guidée

Chaque module vous permettra d'implémenter une partie spécifique du projet avec des instructions détaillées. Vous maîtriserez progressivement l'API First, les tests, le déploiement et le monitoring.
